⚙️ Installation & Setup (Django Project)
1️⃣ Clone Repository
git clone https://github.com/palekarmanshri/Organic-HubX.git
cd Organic-HubX
2️⃣ Create Virtual Environment
python -m venv venv

Activate it:

Windows:
venv\Scripts\activate
Mac/Linux:
source venv/bin/activate
3️⃣ Install Dependencies
pip install django
4️⃣ Run Server
python manage.py runserver

👉 Open in browser:

http://127.0.0.1:8000/
